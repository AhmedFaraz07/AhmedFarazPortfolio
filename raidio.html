Overview

This project was engineered for the Hack Without Borders 2026 event. The goal was to create a robust system capable of providing offline AI access utilizing analog radio protocols.

System Architecture

By leveraging Slow-Scan Television (SSTV) protocols, the system allows for data transmission in environments completely devoid of standard internet infrastructure. The backend logic is driven by Python, interfacing directly with the Gemini API to process and format the transmitted data.
